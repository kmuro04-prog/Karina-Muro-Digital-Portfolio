# Karina Muro - Digital Teacher Portfolio

This repository contains Karina Muro's final digital teacher portfolio for GitHub Pages.

## Files

- `index.html` - the full standalone portfolio website. All portfolio documents are embedded directly in the HTML.
- `README.md` - this setup note.

## GitHub Pages setup

1. Create a new GitHub repository.
2. Upload `index.html` and `README.md` to the root of the repository.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then click **Save**.
6. Wait a minute or two for GitHub to publish the site.
